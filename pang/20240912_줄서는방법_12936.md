## 코드

효율성 시간초과

```java
class Solution {
    int N;
    Long K;
    boolean[] selected;
    int cnt = 0;
    int[] answer;

    public int[] solution(int n, long k) {
        N = n;
        K = k;
        selected = new boolean[n + 1];
        answer = new int[n];
        recFunc(0);
        return answer;
    }

    private void recFunc(int k) {
        //
        if (cnt == K) {
            return;
        }

        if (k == N) {
            cnt++;
            return;
        }

        for (int i = 1; i <= N; i++) {
            if (!selected[i] && cnt != K) {
                selected[i] = true;
                answer[k] = i;
                recFunc(k + 1);
                selected[i] = false;
            }
        }
    }
}
```
